Novas Digitalis API Routes

Landing
HTTP Methods Path Description
GET / Fetches landing/home page

Users
HTTP Method Path Description
GET /users Fetches all Users data from the database
GET /users/:id Fetches a single User's data from the database for the id provided
POST /users/register Adds a new user to the database.
PATCH /users/update/:id Updates a user for the id value provided.
DELETE /users/delete/:id Deletes a user's data from the database for the id provided

Products
HTTP Method Path Description
GET /items Fetches all Products data from the database
GET /items/:id Fetches a product's data from the database for the id provided.
GET /items/recent Fetches the last 4 products added to the database.
POST /items/addProduct Adds a new product to the database.
PATCH /items/update/:id Updates a product for the id value provided.
DELETE /items/delete/:id Deletes a product's data from the database for the id provided

Cart
HTTP Method Path Description
GET /users/:id/carts Fetches all cart data from the database
POST /users/:id/cart Adds items to cart table.
PATCH /users/:id/cart/oid Updates the quantity of items in the cart table.
DELETE /users/:id/cart Clears the cart table.
DELETE /users/:id/cart/:oid Deletes a specified product from the cart table.
